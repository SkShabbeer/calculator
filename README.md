# 🧮 Calculator

A responsive calculator built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations and includes keyboard support, calculation history, and a clean user interface.

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear display
- ⌫ Backspace support
- ⌨️ Keyboard input support
- 📜 Calculation history
- ✨ Animated heading
- 🔐 Secret code feature
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
